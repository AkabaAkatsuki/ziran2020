Welcome to IQ-MPM!

This code distribution contains 6 3D demos as follows:
3D: bear_bath
    balls
    flush_rubber
    dam_jello_ball
    bunny

Instructions for Use:
1. Make sure to download all needed files and place them in either Data/TetMesh (for .mesh) or Data/LevelSets (for .vdb)
2. Compile in build directory as "make coupling -j8"
3. To run your selected demos, run "./coupling -test 1" for the first demo.
